Vector(Matrix) Derivatives

Posted on 2020-01-07 Edited on 2021-04-17 In Tech , Machine Learning

Matrix/vector manipulation

You should be comfortable with these rules. They will come in handy when you want to simplify an expression before differentiating

  • (AB)T=BTAT(AB)^T = B^TA^T(AB)T=BTAT
  • (aTBc)T=cTBTa(a^TBc)^T = c^TB^Ta(aTBc)T=cTBTa
  • aTb=bTaa^Tb = b^TaaTb=bTa (the result is a scalar, and the transpose of a scalar is itself)
  • (A+B)C=AC+BC(A + B)C = AC + BC(A+B)C=AC+BC (multiplication is distributive)
  • (a+b)TC=aTC+bTC(a + b)^T C = a^TC + b^T C(a+b)TC=aTC+bTC
  • AB=/ BAAB {=}\mathllap{/\,} BAAB=/BA (multiplication is not commutative)

Derivatives of Matrices, Vectors and Scalar Forms

First-order(Linear Products)

Let f=xTaf = x^Taf=xTa, compute dfdx\frac{df}{dx}dxdf​

Solution

df=d(xT)a=d(x)Ta=aTdx→dfdx=adf = d(x^T)a = d(x)^Ta = a^Tdx \to \frac{df}{dx} = a df=d(xT)a=d(x)Ta=aTdx→dxdf​=a

We also have d(aTx)dx=a\frac{d(a^Tx)}{dx} = adxd(aTx)​=a

Let f=aTXbf = a^TXbf=aTXb, compute dfdX\frac{df}{dX}dXdf​

Solution

using AB=BAAB = BAAB=BA when AB is a scalar,

df=aTdXb=baTdX→dfdX=abTdf = a^Td{X}b = ba^TdX \to \frac{df}{dX} = ab^T df=aTdXb=baTdX→dXdf​=abT

Let f=aTXTbf = a^TX^Tbf=aTXTb, compute dfdX\frac{df}{dX}dXdf​

Solution

df=aTd(XT)b=aTd(X)Tb=(d(X)a)Tb=bTd(X)a=abTd(X)→dfdX=baTdf = a^Td(X^T)b = a^Td(X)^Tb = (d(X)a)^Tb = b^Td(X)a = ab^Td(X) \to \frac{df}{dX} = ba^T df=aTd(XT)b=aTd(X)Tb=(d(X)a)Tb=bTd(X)a=abTd(X)→dXdf​=baT

Second-order(Quadratic Products)

Solution

df=bTd(XT)Xc+bTXTd(X)c(1)\tag{1} df = b^Td(X^T)Xc + b^TX^Td(X)cdf=bTd(XT)Xc+bTXTd(X)c(1)

From (1)

bTd(XT)Xc=(Xc)Td(X)b=b(Xc)Td(X)=bcTXTd(X)=(XcbT)Td(X)(2)\tag{2} b^Td(X^T)Xc = (Xc)^Td(X)b = b(Xc)^Td(X) = bc^TX^Td(X) = (Xcb^T)^Td(X) bTd(XT)Xc=(Xc)Td(X)b=b(Xc)Td(X)=bcTXTd(X)=(XcbT)Td(X)(2)

bTXTd(X)c=cbTXTd(X)=(XbcT)Td(X)(3)\tag{3} b^TX^Td(X)c = cb^TX^Td(X) = (Xbc^T)^Td(X) bTXTd(X)c=cbTXTd(X)=(XbcT)Td(X)(3)

Replace (2),(3) into (1)

df=((Xcb)T+(XbcT)T)d(X)→dfdX=X(cbT+bcT)df = ((Xcb)^T + (Xbc^T)^T)d(X) \to \frac{df}{dX} = X(cb^T + bc^T) df=((Xcb)T+(XbcT)T)d(X)→dXdf​=X(cbT+bcT)
